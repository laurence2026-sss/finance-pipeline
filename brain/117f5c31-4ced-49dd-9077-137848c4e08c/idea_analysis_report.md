## 5. 🔍 숨은 정부지원금 알리미 봇 (Gov-Hunter)

### 1-1. 🎯 시장 분석 (Market Overview)
*   **규모**: 매년 수조 원 대의 정부 예산이 집행되지만, 홍보 부족으로 미집행되는 예산이 상당함. '몰라서 못 받는 돈'에 대한 대국민적 관심은 언제나 1위.
*   **고통 포인트(Pain Point)**:
    *   정책이 너무 많고 각 부처별로 흩어져 있음.
    *   공고문이 복잡함 (수십 페이지 HWP 파일).
    *   신청 기간을 놓치는 경우가 다반사.

### 1-2. ⚔️ 경쟁사 분석 (Competitive Landscape)
*   **토스(Toss), 각종 핀테크 앱**: '내 모든 혜택 찾기' 기능 등 대기업이 이미 진출.
    *   *약점*: 너무 포괄적임. "나에게 딱 맞는" 초개인화된 *알림* 기능은 부족할 수 있음.
*   **웰로(Wello)**, **정책브리핑**: 전문적인 정책 큐레이션 앱.
    *   *약점*: 앱 설치의 장벽. UI가 다소 복잡함.
*   **차별화 전략**: **"초간단 & 노티피케이션 중심"**
    *   앱 설치 X → 카카오톡 알림톡/문자로 핵심만 전달.
    *   복잡한 공고문 대신 "3줄 요약 + 신청 링크"만 제공.

### 1-3. 📝 상세 계획서 (Execution Plan)
1.  **데이터 수집**: 공공데이터포털(data.go.kr) API 활용 + 지자체 공지 크롤링.
2.  **프로필 입력**: 나이/거주지/소득분위/직업/자녀유무 등 5가지 핵심 정보만 입력받음.
3.  **매칭 알고리즘**: 조건에 맞는 공고가 뜨면 즉시 알림 발송.
4.  **수익 모델**:
    *   초기: 무료 (트래픽 확보).
    *   중기: 금융 상품 추천 (적금/대출) 제휴.
    *   후기: "신청 대행 서비스" (유료 프리미엄).

## 6. 🐾 내 반려동물 AI ID 카드 (Pet Passport)

### 2-1. 🎯 시장 분석 (Market Overview)
*   **규모**: 국내 반려동물 양육 인구 1,500만 명 ("펫팸족"). 펫 시장 규모 4조 원 돌파.
*   **트렌드**: 반려동물을 가족처럼 여기며(Humanization), 이들을 위한 굿즈, 사진, 추억 남기기에 돈을 아끼지 않음.
*   **고통 포인트(Pain Point)**:
    *   단순한 사진 굿즈는 식상함.
    *   동물 등록증 발급이 번거롭고 디자인이 투박함.
    *   "내 새끼"를 귀엽게 자랑하고 싶은 욕구는 강하지만 콘텐츠 생산 능력이 부족함.

### 2-2. ⚔️ 경쟁사 분석 (Competitive Landscape)
*   **아이디어스/스마트스토어 주문 제작**: 커스텀 등록증 판매 업체 다수.
    *   *약점*: 사진을 보내고 제작→배송까지 3~7일 소요. 디자인이 정해져 있음.
*   **AI 프로필 서비스 (스노우 등)**: 사람 위주. 동물 전용은 아직 틈새시장.
*   **차별화 전략**: **"즉시성 & 재미(Fun)"**
    *   기다릴 필요 없음. 사진 올리면 3초 만에 고퀄리티 여권/ID카드 이미지 생성.
    *   "개통령", "츄르 감별사" 같은 재미있는 직함/문구 자동 생성.

### 2-3. 📝 상세 계획서 (Execution Plan)
1.  **이미지 생성**: 사용자가 업로드한 반려동물 사진의 배경을 제거(Remove.bg) + 생성형 AI(Stable Diffusion/DALL-E)로 '제복 입은 강아지', '여행가는 고양이' 등 합성.
2.  **템플릿 합성**: 여권, 운전면허증, 사원증 등 다양한 템플릿에 사진과 정보(이름, 생일, 특징) 합성.
3.  **바이럴 유도**: 결과물에 "이 여권으로 츄르국 입국 가능" 같은 문구를 넣어 인스타 스토리 공유 유도.
4.  **수익 모델**:
    *   기본 이미지 무료 (워터마크 포함).
    *   고화질 이미지 + 워터마크 제거: 1,000원.
    *   실물 플라스틱 카드 배송: 15,000원 (POD 인쇄소 연동).
